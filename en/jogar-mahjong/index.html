How to play this version

This is a Mahjong Solitaire version. The goal is to clear the board by matching pairs of identical tiles. To select a tile, it must be free: no tile can be on top of it, and at least one side must be open.

Step by step

  1. Look for two tiles with the same symbol.
  2. Check if both are free. Blocked tiles appear faded.
  3. Click the first tile and then the second.
  4. When you form a correct pair, both leave the board.
  5. Continue until you clear everything or no moves remain.

Tips to win more games

Start by removing tiles from the upper layers because they block other tiles. Then try to free long rows on the sides. If there are four identical tiles available, think about which pair opens more space before clicking.

Use the Hint button when you want to find a free pair. Use Shuffle only when the board gets stuck, because part of Mahjong's fun is figuring out the best sequence yourself.
