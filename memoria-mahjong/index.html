Como jogar Memória Mahjong

Memória Mahjong é uma versão tranquila e divertida do clássico jogo da memória, usando peças tradicionais de Mahjong. É perfeito para relaxar e treinar a memória ao mesmo tempo.

Regras

  1. Clique em uma peça para virá-la e ver qual símbolo está escondido.
  2. Clique em outra peça para tentar encontrar o par igual.
  3. Se as duas peças forem iguais, elas ficam viradas para cima.
  4. Se forem diferentes, elas viram de volta depois de um momento.
  5. Continue até encontrar todos os pares.

Dicas para melhorar

Tente memorizar a posição das peças mesmo quando elas não formam um par. Com o tempo, você vai reconhecendo padrões e melhorando seu tempo. Quanto menos movimentos, melhor!
