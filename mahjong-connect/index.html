Como jogar Mahjong Connect

Mahjong Connect é uma variação divertida do clássico jogo de combinar peças. Aqui o objetivo também é encontrar pares iguais, mas com uma regra especial de conexão.

Regras

  1. Procure duas peças com o mesmo símbolo no tabuleiro.
  2. Verifique se é possível conectar as duas com uma linha reta que faz no máximo dois cantos.
  3. A linha não pode passar por cima de outras peças.
  4. Clique nas duas peças para removê-las.
  5. Continue até limpar todo o tabuleiro.

Dicas para vencer

Comece pelas peças das bordas e dos cantos, pois elas têm menos caminhos bloqueados. Preste atenção em peças que estão "presas" no meio de outras e tente liberá-las primeiro. Use o botão Dica quando estiver travado.
